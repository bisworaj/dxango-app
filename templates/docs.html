Follow along :-

1. Install django

pip install django 
C:\Users\bisworajSaheb>pip install django

this should install django web framework on your machie. before that make sure already have installed python on your machine

2. Go to your worksapce & create your first django project

django-admin startproject project_name 
                C:\Users\bisworajSaheb>cd workspace
                C:\Users\bisworajSaheb\Workspace>django-admin startproject dxango

this should create a compact django webapp setup on that folder.in my case the django project is "dxango"

3. Create Your first app inside your django project

django-admin startapp app_name 
                C:\Users\bisworajSaheb\Workspace\dxango>
                C:\Users\bisworajSaheb\Workspace\dxango>django-admin startapp home

This command should create one app inside your django project folder. in may case app name is "home".

4. Run django server to view your site in browser.

python manage.py runserver 
                C:\Users\bisworajSaheb\Workspace\dxango>python manage.py runserver

this should start django server on your machine. just follow the link given to view it.
